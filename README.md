# BI-CSP IDE Scrapper
PowerBI scrapper scritp for IDE database colection from BI-CSP

## cheat sheet

### venv
create venv
```bash
python3 -m venv .venv
```

activate venv
```bash
source .venv/bin/activate
```

### Docker
build docker image
```bash
docker build -t main:latest .
```

### Chromedriver
```
brew install chromedriver
```
